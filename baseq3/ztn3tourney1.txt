* Play Information *

- Make sure you have the latest Quake III: Arena patch installed.
  For more information, visit http://www.quake3arena.com/
- Unzip the contents of ztn3tourney1.zip to your Quake3\baseq3 folder
- Run Quake III: Arena 
- From the main menu select Single player then Skirmish. The map
  should appear somewhere on the list, usually at the end.


* Description *

Game type     : Tourney
Respawn spots : 9

What's changed: 
- Quad Damage removed
- Clipped out the ceiling area near the Lightning Gun
